# AI Email Security System

## Overview

AI Email Security System is a machine learning-based application that detects spam and phishing emails automatically. The system analyzes email content using Natural Language Processing (NLP) techniques and classifies messages as spam or safe.

## Features

* Spam email detection using a Naive Bayes model
* NLP text processing with TF-IDF
* Suspicious link detection
* Phishing keyword identification
* Email risk level analysis
* Interactive web dashboard using Streamlit

## Technologies Used

* Python
* Scikit-learn
* Pandas
* Streamlit
* Matplotlib

## How to Run

1. Install required libraries:
   pip install streamlit pandas scikit-learn matplotlib

2. Run the application:
   python -m streamlit run ai_email_security.py

3. Open the browser link generated by Streamlit and test email messages.
4. ## Features

- Spam email detection using Machine Learning
- Phishing keyword detection
- Suspicious link detection
- Email risk level analysis
- Interactive Streamlit dashboard

## Project Goal

The goal of this project is to demonstrate how machine learning and NLP can be used to improve email security by detecting spam and phishing attempts.
